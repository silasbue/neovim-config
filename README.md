# neovim-config

### delete old config

```zsh
rm -rf ~/.config/nvim ~/.local/share/nvim ~/.local/state/nvim ~/.cache/nvim
```

### install this config

```zsh
# ssh
git clone git@github.com:silasbue/neovim-config.git ~/.config/nvim

# https
git clone https://github.com/silasbue/neovim-config.git ~/.config/nvim
```
